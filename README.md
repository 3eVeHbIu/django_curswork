Курсовая работа по высокоуровнему языку программированя Python.

Простой новостной сайт с использованием фреймоворка Django.
На базе данных MySql. 
Имеюстся страницы регистрации (проводится верификация), авторизации (проводится верификация), восстановления пароля. 
Используется captcha.
Доступ к основной странице сайта ограничен всем кто не авторизирован. Создавать новости может любой пользователь, редактировать можно только свои.
